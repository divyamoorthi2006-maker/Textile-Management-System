# Textile Management System

A simple **Java Console-Based Textile Management System** that helps manage textile products and inventory. This project is developed using **Core Java** and demonstrates Object-Oriented Programming (OOP) concepts such as classes, objects, constructors, and collections.

## 📌 Features

- ➕ Add new textile products
- 📋 View all available products
- 🔍 Search product by Product ID
- 💰 Calculate total inventory value
- 🚪 Exit the application

## 🛠️ Technologies Used

- Java
- ArrayList
- Scanner Class
- Object-Oriented Programming (OOP)

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Textile-Management-System.git
```

### 2. Open the project

Open the project in any Java IDE such as:
- IntelliJ IDEA
- Eclipse
- NetBeans
- VS Code

### 3. Compile

```bash
javac TextileManagementSystem.java
```

### 4. Run

```bash
java TextileManagementSystem
```
